You have received a suspicious Windows executable file named suspicious.exe, Without
running it, identify unique patterns or signatures that could indicate its malware family or
type. Use an open-source tool to create rules or signatures that can help classify this file

Tool: YARA

Important Disclaimer: We will NOT use live malware binaries. That would be unsafe and
unethical. Instead, we use malware signatures embedded into test binaries.

The main goal is to

YARA is an open-source tool used to identify and classify malware based on
textual patterns, strings, and binary characteristics.
By performing static analysis, malware behavior can be inferred without
running the file, ensuring system safety.
What are we Analyzing:
We analyze binary files (mostly .exe) and look for:

п‚· Windows API function names
п‚· PE(Portable Executable) file headers
